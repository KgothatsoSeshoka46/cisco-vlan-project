# Multi-LAN Network Design - Cisco Packet Tracer

## Objective

Designed and implemented a network with two separate LANs connected through a router to enable inter-network communication.

## Network Design

* 2 Local Area Networks (LANs)
* Each LAN contains:

  * 3 PCs
  * A switch
* A router connects both LANs

## Configuration Details

* Manually assigned IP addresses to all PCs
* Configured router interfaces for each network
* Set default gateways on all devices
* Verified connectivity using ping between different LANs

## Features

* Communication within each LAN
* Communication between different LANs via router
* Basic routing configuration

## Technologies Used

* Cisco Packet Tracer
* IP Addressing
* Routing fundamentals

## What I Learned

* How routers connect multiple networks
* Difference between a switch and a router
* Importance of default gateways
* Basic network troubleshooting using ping

## Future Improvements

* Implement VLANs for better network segmentation
* Add DHCP for automatic IP assignment
* Apply network security (ACLs)
